About This Project

This web-based system is designed to help users detect plant diseases from uploaded images using an AI-powered model. It combines machine learning, Django, and Bootstrap to deliver accurate predictions and practical guidance for farmers, students, and agricultural researchers.

üåø Key Features

  • Image Upload & Preview: Upload plant leaf/fruit images and preview them before classification.
  • AI-Powered Prediction: A TensorFlow model trained using Google Teachable Machine predicts the plant condition with a confidence score.
  • Disease Information: Displays causes, symptoms, treatment, and prevention tips based on prediction.
  • Low Confidence Warning: If confidence is below 90%, users are alerted to double-check the result.
  • Database Storage: Stores prediction history in an SQLite database for tracking and analysis.
  • Responsive UI: Clean, mobile-friendly interface built with Bootstrap 5.
  • Backend API: A POST endpoint handles prediction and storage securely with CSRF protection.
